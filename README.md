# Smart Queue Management System

Smart Queue Management System is a backend application built with Go and the Gin framework.  
It provides APIs for managing business queues and delivers real-time updates using Server-Sent Events (SSE).  
The system helps businesses track customer flow and keep users informed about their queue status instantly.

---

## Features

- Business-wise queue management  
- Customers can join or leave a queue  
- Real-time queue updates using SSE  
- Automatic database migration with GORM  
- Clear and organized project structure
